Overview
--------

This repository contains code used to quantify forest canopy responses to the 2021 Pacific Northwest heat dome at Ellsworth Creek Preserve, Washington, USA. The analysis focuses on detecting canopy stress, identifying structural and management drivers of vulnerability, and evaluating post-disturbance recovery using high-resolution remotely sensed NDVI dervied from PlanetScope.
